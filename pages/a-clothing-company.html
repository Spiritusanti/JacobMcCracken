React + Redux + Firebase + Stripe

Main Project of the Complete React Developer in 2021(w/ Redux, Hooks, GraphQL) from the Zero to Mastery Academy

The project was built using Create-React-App, Redux, Styled Components, Google Firebase, and accepts payment via Stripe API

Project facilitated a deep dive into the React ecosystem from basic syntax to implementing React Router, Redux, and Firebase with hosting via Heroku

This is the most complex project I have built to date with React and I found it challenging to keep up with its complexity. The most challenging part was implementing Firebase and linking it up to facilitate storing/creating/accessing user info and authentication both through Google's auth service as well as the custom authentication
